# RxExtract - Android Application

RxExtract is a modern prescription data extraction and workspace manager designed for Android. It enables users to scan prescription images, extract medical information efficiently (using advanced Gemini AI vision models), organize them into dedicated workspaces, and export the structured results.

## Features
- **Prescription Scan**: Capture or load prescription images to extract medicine names, dosages, durations, departments, and counts.
- **Workspaces**: Group prescription scans into custom-created workspaces for better organization and collaborative or thematic grouping.
- **Local History**: Browse local extraction records and manage saved data completely offline.
- **Structured Export**: Download and export extracted prescription data into CSV formats.
- **Adaptive UI**: Formatted elegantly following Material Design 3 guidelines.

## Technologies Used
- **Kotlin & Jetpack Compose**: High-performance, modern declarative UI building blocks.
- **Room Database**: Safe and structured local persistence for workspace and extraction history records.
- **Coroutines & Flow**: Efficient asynchronous operations and real-time state management.
- **Material Design 3 (M3)**: Beautiful, responsive visual presentation with full support for dark and light modes.
